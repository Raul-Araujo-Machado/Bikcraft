# Bikcraft
 Projeto fictício de um site de vendas de biciletas elétricas personalizaveis. A construção foi realizada desde os wireframes até o protótipo final de alta fidelidade no curso de UI Design. E foi desenvolvido como projeto final do curso de HTML e CSS.
